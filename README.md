# WHEEL

A simple web application to view and download information.

## Features

- Displays data records in a table
- Download information as **JSON** or **CSV**

## Usage

Open `index.html` in a browser (or serve it via any static file server):

```bash
npx serve .
```

Then click the **Descargar JSON** or **Descargar CSV** button to download the data.

## Files

| File | Description |
|------|-------------|
| `index.html` | Main webpage with table and download buttons |
| `app.js` | Loads data and handles download logic |
| `data.json` | Sample data records |
